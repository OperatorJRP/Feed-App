# Feed App

Minimal social feed prototype powered by React, TypeScript, and Vite. Uses Zustand for state, React Router for navigation, and plain CSS for styling. All data is stored locally via `localStorage`.

## Quick Start
```bash
npm install
npm run dev
```

## Scripts
- `npm run dev` — develop
- `npm run build` — production build
- `npm run preview` — preview build
- `npm run lint` — lint source
- `npm run format` — format code

## Structure
```
src/
```
